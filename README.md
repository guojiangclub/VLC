# vlc-ui

## 一、说明

> 一套基于Vue 2.0的通用组件库。


## 二、安装

> 执行以下命令，在你的项目中引入组件库
```
    npm install vlc-ui
```

> 引入之后，在你项目的入口通过import引入组件库

```
import Vue from 'vue'
import Vlcui from 'vlc-ui'

Vue.use(Vlcui)

```


> 在你项目的入口vue文件中引入组件库样式


```
// example in app.vue

<template>
    // 省略部分html代码
</template>

<style>

    @import "~vlc-ui/dist/styles/vlc.css"
</style>

```

> 现在可以开始组件库的使用了。
